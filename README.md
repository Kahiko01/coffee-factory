# Karuhiu Utheri Digital Farmers Cooperative


## Overview
Digital coffee factory management system for Kenyan farmers.

## Quick Start
```bash
git clone https://github.com/Kahiko01/coffee-factory.git
cd coffee-factory
pip install -r requirements.txt
python3 populate_data.py
python3 app.py
```

## Features
- 3D Coffee Plant Visualization
- Payment Management with PDF Statements
- Analytics Dashboard with Charts
- AI Agronomist Copilot
- 100 Pre-loaded Farmers
- Role-based Access Control

## Tech Stack
Flask | SQLAlchemy | Chart.js | Three.js | ReportLab
